## Примеры обработки

### Пример 1: Циркон

**Ввод:**
```
Мороков циркон на винте 7шт пациент Иванов
```

**Вывод:**
```
Техник: Мороков
Пациент: Иванов
Вид работы: коронка из диоксида циркония на винте
Количество: 7 шт
```

### Пример 2: Металлокерамика

**Ввод:**
```
Сидоров металлокерамика 13шт на завтра пациент Петров
```

**Вывод:**
```
Техник: Сидоров
Пациент: Петров
Вид работы: металлокерамическая коронка
Количество: 13 шт
Заметки: на завтра
```

### Пример 3: Виниры

**Ввод:**
```
Козлов виниры 5шт от Иванова 15.02.2026 пациент Сидоров
```

**Вывод:**
```
Техник: Козлов
Врач: Иванов
Пациент: Сидоров
Вид работы: керамические виниры из диоксида циркония
Количество: 5 шт
Срок: 15.02.2026
```
