## Найденные проблемы

### 1. ✅ Неверный параметр timeout
**Проблема:**
```
TypeError: Client.__init__() got an unexpected keyword argument 'proxies'
```

**Причина:**
Библиотека `openai` изменила API. Параметры `timeout` больше не поддерживаются в конструкторе клиента.

**Решение:**
Убраны параметры `timeout` из конструкторов OpenAI в обоих файлах:
- `src/services/dental_terminology_service.py`
- `src/services/message_processor.py`

### 2. ✅ Ошибки escape-последовательностей
**Проблема:**
```
SyntaxError: f-string: single '}' is not allowed
SyntaxWarning: "{" is an invalid escape sequence
```

**Причина:**
Мой скрипт для фиксации скобок добавил неправильные escape-последовательности:
- `{{` вместо `{`
- `\{{` вместо `{`
- `}}` вместо `}`
- `\\{` вместо `{`

**Решение:**
Восстановлен оригинальный код без неправильных escape-последовательностей:
- Убраны все двойные скобки `{{` и `}}`
- Убраны все escape-последовательности `\{`, `\}`, `\\{`

## Как обновить на сервере

### Быстрый способ (ОДНА КОМАНДА):

```bash
scp src/services/dental_terminology_service.py src/services/message_processor.py user@server:/opt/basestom-bot/src/services/ && ssh user@server "cd /opt/basestom-bot && sudo supervisorctl restart basestom-bot"
```

### Пошаговый способ:

#### Шаг 1: Подключение к серверу
```bash
ssh user@your-server-ip
```

#### Шаг 2: Переход в директорию
```bash
cd /opt/basestom-bot/src/services
```

#### Шаг 3: Создание бекапов
```bash
cp dental_terminology_service.py dental_terminology_service.py.backup
cp message_processor.py message_processor.py.backup
```

#### Шаг 4: Проверка синтаксиса
```bash
python -m py_compile dental_terminology_service.py
python -m py_compile message_processor.py

echo "Синтаксис проверен успешно!"
```

#### Шаг 5: Перезапуск бота
```bash
# Supervisor
sudo supervisorctl restart basestom-bot

# Проверить статус
sudo supervisorctl status basestom-bot

# Посмотреть логи
sudo tail -f /var/log/supervisor/basestom-bot.log
```
