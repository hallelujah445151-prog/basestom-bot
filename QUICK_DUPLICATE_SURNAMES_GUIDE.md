# КРАТКОЕ РУКОВОДСТВО: НОВАЯ ЛОГИКА ПОИСКА

## Что изменилось

### Принцип работы:
1. **Уникальная фамилия** → поиск работает и по фамилии, и по полному имени
2. **Дублирующиеся фамилии** → поиск ТОЛЬКО по полному имени

---

## Примеры использования

### Уникальная фамилия (нет дубликатов):
```
✅ По фамилии: "Мороков циркон на винте 7шт"
   Результат: НАЙДЕН ✅

✅ По полному имени: "Мороков Александр Александрович циркон на винте 7шт"
   Результат: НАЙДЕН ✅

✅ Fuzzy: "Мороков А.А. циркон на винте 7шт"
   Результат: НАЙДЕН ✅
```

### Дублирующиеся фамилии:
```
❌ По фамилии: "Мороков циркон на винте 7шт"
   Результат: НЕ НАЙДЕН ❌

✅ По полному имени: "Мороков Александр Александрович циркон на винте 7шт"
   Результат: НАЙДЕН ✅

❌ Fuzzy: "Мороков А.А. циркон на винте 7шт"
   Результат: НЕ НАЙДЕН ❌
```
