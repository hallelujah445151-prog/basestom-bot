## 🔧 Если тест не работает

### Проблема 1: OpenRouter API не отвечает
**Решения:**
1. Проверьте интернет: `ping google.com`
2. Проверьте VPN: откройте сайт OpenRouter
3. Попробуйте другой ключ API (получите новый на https://openrouter.ai/keys)

### Проблема 2: Скрипт не запускается
**Решения:**
1. Откройте папку `basestom` в проводник
2. Дважды кликните на `test_openrouter.py` → "Открыть с помощью Python"
3. Или запустите из PowerShell:
   ```powershell
   cd C:\Users\crush\AppData\Roaming\projects\basestom
   python test_openrouter.py
   ```
