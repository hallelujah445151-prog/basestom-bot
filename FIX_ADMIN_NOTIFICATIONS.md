# Исправление проблемы с уведомлениями администраторам

## Проблема
При создании заказа уведомление приходило только:
- Технику, назначенному в заказе
- Врачу (если указан)
- Создателю заказа (админу, который нажал /neworder)

Другие администраторы НЕ получали уведомления.

## Решение
Добавлена отправка уведомлений всем администраторам при создании заказа.

## Изменения

### 1. NotificationService (notification_service.py)
Добавлен метод `send_to_all_admins()`:
- Находит всех администраторов в базе данных
- Отправляет уведомление каждому администратору
- Включает фото заказ-наряда
- Логирует успех/неудачу отправки

### 2. Orders Handler (orders.py)
Добавлен вызов `send_to_all_admins()` при создании заказа:
```python
await notification_service.send_to_all_admins(order_data, photo_id)
```

## Формат уведомления администраторам
```
🔔 Новый заказ в системе!

🆔 Заказ №123
👤 Пациент: Иванов
🔧 Техник: Мороков
👨‍⚕️ Врач: Петров
🔨 Вид работы: цельноцирконевая коронка на винте
📊 Количество: 7 шт
📅 Срок выполнения: 15.04.2026
```

## Результат
✅ Теперь ВСЕ администраторы получают уведомления о новых заказах
✅ Техник получает уведомление о назначении
✅ Врач получает уведомление о заказе
✅ Создатель заказа получает подтверждение
✅ Включается фото заказ-наряда для всех получателей

## Логирование
- Показывает количество найденных администраторов
- Показывает успешную отправку каждому администратору
- Показывает статистику отправки (успешно/всего)
