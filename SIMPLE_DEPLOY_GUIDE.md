## Как деплоить на сервер (Linux)

### 1. Загрузить файлы на сервер
```bash
# Через SCP
scp -r basestom/ user@server:/opt/

# Или через git
git clone <your-repo-url> /opt/basestom
cd /opt/basestom
```

### 2. Настроить .env файл
```bash
# Создать файл .env
nano .env

# Или скопировать пример
cp .env.example .env
nano .env
```

### 3. ОДНИМ КЛИКОМ - деплой
```bash
bash deploy_simple.sh
```

ВСЁ! Бот запустится.

---

## Как деплоить локально (Windows)

### 1. Настроить .env файл
```
Создать файл .env с:
BOT_TOKEN=your_bot_token
OPENROUTER_API_KEY=your_api_key
```

### 2. ОДНИМ КЛИКОМ - запуск
```
Дважды кликнуть на: deploy_simple.bat
```

ВСЁ! Бот запустится.

## Что делают простые скрипты

### deploy_simple.sh (Linux)
1. ✅ Проверяет наличие .env файла
